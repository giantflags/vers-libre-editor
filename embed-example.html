Vers Libre Editor - Embedded Version

This demonstrates how to embed the editor in your website

How to Embed in Google Sites:

Method 1: Simple Embed

1. In Google Sites, click Insert → Embed

2. Paste this URL:

https://giantflags.github.io/vers-libre-editor/?embed=true

3. Set height to 800px or more

Method 2: Custom HTML

Use this HTML code in an embed block:

<iframe src="https://giantflags.github.io/vers-libre-editor/?embed=true" width="100%" height="900px" frameborder="0" allowfullscreen title="Vers Libre Image Editor"> </iframe>
